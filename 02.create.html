Creating a Database

To create a database, we will start by creating a MongoClient object. Then we will specify a connection URL with the correct IP address and the name of the database we want to create.

If the database doesn't exist, it will be created and a connection will be established:


      // app.js

      var MongoClient = require('mongodb').MongoClient;
      var url = "mongodb://localhost:27017/mydb";

      MongoClient.connect(url, function(err, db) {
        if (err) throw err;
        console.log("Database created!");
        db.close();
      });

Run the app.js file containing the code above using the following:


      node app.js

      // Result: Database Created!
