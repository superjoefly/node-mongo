Creating Collections

Collections in MongoDB are similar to tables in MySQL.

We can use the createCollection() method to create a collection in MongoDB.


      // app.js

      var MongoClient = require('mongodb').MongoClient;
      var url = "mongodb://localhost:27017/mydb";

      MongoClient.connect(url, function(err, db) {
        if(err) throw err;
        db.createCollection("customers", function(err, res) {
          if(err) throw err;
          console.log("Collection Created!");
          db.close();
        });
      });

Run the app.js file containing the code above using the following:


      node app.js

      // Result: Collection Created!
