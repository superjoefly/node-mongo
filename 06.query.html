MongoDB Query

Filtering Results

We can use a query object to filter results when retrieving documents from a collection.

The first argument of the find() method is a query object, and can be used to limit our search:


      var MongoClient = require('mongodb').MongoClient;
      var url = 'mongodb://localhost:27017/mydb';

      MongoClient.connect(url, function(err, db) {
        if(err) throw err;
        var query = {address: '226 Left St'};
        db.collection('customers').find(query).toArray(function(err, result) {
          if(err) throw err;
          console.log(result);
          db.close();
        });
      });

Run the app.js file containing the code above using the following:


      node app.js

      // Result:
          // [ { _id: 59ee910440030506aa61d276,
          // name: 'Rebecca',
          // address: '226 Left St' } ]

Filter Using Regular Expression

We can also write regular expressions to find exactly what we are looking for.

Regular expressions are limited to querying strings.

Retrieve all documents where the address field starts with the letter B:


      var MongoClient = require('mongodb').MongoClient;
      var url = 'mongodb://localhost:27017/mydb';

      MongoClient.connect(url, function(err, db) {
        if(err) throw err;
        var query = { address: /^B/ };
        db.collection('customers').find(query).toArray(function(err, result) {
          if(err) throw err;
          console.log(result);
          db.close();
        });
      });

Run the app.js file containing the code above using the following:


      node app.js

      // Result
          // [ { _id: 59ee477f406931606fb2a004,
          //     name: 'My Company',
          //     address: 'Boulder CO' } ]
