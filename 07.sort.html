Sort

Sorting Results

We can use the sort() method to sort results into ascending or descending order.

The sort() method takes one argument that holds an object defining the sorting order.

Using the value 1 in the sort object will sort results in ascending order.

Using the value -1 in the sort object will sort results in descending order.

// { name: 1 } = ascending

// { name: -1 } = descending

Sort - Ascending

The following example sorts the results using the name field in alphabetical order:


      // Ascending Order

      var MongoClient = require('mongodb').MongoClient;
      var url = 'mongodb://localhost:27017/mydb';

      MongoClient.connect(url, function(err, db) {
        if(err) throw err;
        var mysort = { name: 1 };
        db.collection('customers').find().sort(mysort).toArray(function(err, result) {
          if(err) throw err;
          console.log(result);
          db.close();
        });
      });

Sort - Descending

The following example will sort the results in reverse alphabetical order:


      // Descending Order

      var MongoClient = require('mongodb').MongoClient;
      var url = 'mongodb://localhost:27017/mydb';

      MongoClient.connect(url, function(err, db) {
        if(err) throw err;
        var mysort = { name: -1 };
        db.collection('customers').find().sort(mysort).toArray(function(err, result) {
          if(err) throw err;
          console.log(result);
          db.close();
        });
      });
