MongoDB Limit

Limiting Results

We can use the limit() method to limit the result of a MongoDB query.

The limit() method only takes one paramter which defines how many documents to return.

In the following example, we will limit the number of documents returned to 3:


      var MongoClient = require('mongodb').MongoClient;
      var url = 'mongodb://localhost:27017/mydb';

      MongoClient.connect(url, function(err, db) {
        if(err) throw err;
        db.collection('customers').find().limit(3).toArray(function(err, result) {
          if(err) throw err;
          console.log(result);
          db.close();
        });
      });

Run the app.js file containing the code above using the following:


      node app.js
